# Depth-Chart Competition Feature ships in v1; trades and coaching/scheme changes are deferred to v2

Team/context features were initially deferred wholesale, but research surfaced that free-agency-driven depth-chart shifts are the single most common reason real preseason projections miss — a trailing-stats-only model has zero signal for "this team just signed a new starter at this position." We're adding one narrowly-scoped feature for this (current-year vs. prior-year positional roster composition per team, mechanically derivable from nflverse roster data, no subjective encoding). Trades (player changes teams) and coaching/scheme changes stay deferred to v2 because they require genuine judgment calls (e.g. "is this a good landing spot?") rather than a clean derived feature — pulling them in now would have re-opened exactly the scope creep this project is trying to avoid in v1.
