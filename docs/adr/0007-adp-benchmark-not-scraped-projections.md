# Evaluate against a naive baseline and historical ADP, not scraped historical projections

We wanted to benchmark against real expert consensus, not just a naive statistical baseline, since "beat the crowd" is the bar that actually matters for draft edge. Research found historical ADP is cleanly available for free (Fantasy Football Calculator API, verified for 2020–2024), but historical *point* projections from major platforms (FantasyPros, ESPN, PFF) are not — no vendor offers a documented free API for past preseason projections, and the `ffanalytics` package's own docs say scraping historical seasons won't work. The only viable route to point projections is Wayback Machine scraping of FantasyPros' pages, which is fragile (per-snapshot HTML scraping, not an API). v1 evaluation uses the naive baseline plus the ADP Benchmark; scraped point projections are a stretch goal, not a requirement. See `docs/research/historical-adp-and-projections-availability.md`.
