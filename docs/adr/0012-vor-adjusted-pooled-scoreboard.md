# Cross-position pooled scoreboard uses VOR-adjusted actuals

Per-position scoreboards need no adjustment — raw total points compare fine within a position. But a single pooled ranking scored against raw actual points would rank QBs artificially high (QB20 out-scores most WR1s in raw points), because raw points ignore that only one QB starts per team and replacement QBs are freely available. Draft value is points **over the replacement-level player at the position**, which is exactly the logic ADP's crowd already prices in.

The pooled Answer Key is therefore VOR-adjusted: actual season total points minus the actual points of that season's Replacement Baseline player at the position. Baselines use the last-starter convention for 12-team PPR (matching the FFC ADP source): QB12, RB24, WR36, TE12, computed retrospectively from each season's actual finishes. Flex-adjusted baselines (RB~30/WR~42) are a knob, not a decision — don't over-engineer this. The per-position scoreboards (ADR-0010, ADR-0011, ADR-0014) are unaffected by this pooled cross-position addition.
